Additionally, recall precision was not greatly affected (note that recall can depend on various factors, such as the [quantization technique](https://docs.opensearch.org/latest/mappings/supported-field-types/knn-memory-optimized/#quantization-techniques) used and the data distribution).

When using `byte` vectors, expect some loss of recall precision compared to using `float` vectors. Byte vectors are useful in large-scale applications and use cases that prioritize a reduced memory footprint in exchange for a minimal loss of recall.
